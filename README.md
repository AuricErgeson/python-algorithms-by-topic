# Python Algorithms by Topic

A comprehensive collection of Python programming exercises from [PYnative.com](https://pynative.com/), organized by topic for systematic learning and practice.

## About This Project

This repository serves as a personal learning journey through Python programming, containing solutions to exercises from PYnative.com. The goal is to build a solid foundation in Python by practicing algorithms and coding challenges organized by specific topics and difficulty levels.

## Purpose

- **Practice:** Hands-on coding to reinforce Python concepts
- **Organization:** Exercises categorized by topic for easy navigation
- **Reference:** A growing collection of solutions for future review
- **Growth:** Continuous learning by adding new topics and challenges

## About PYnative

[PYnative.com](https://pynative.com/) is a comprehensive resource for Python learning that offers:
- Practical coding exercises with solutions
- Python tutorials and guides
- Quizzes to test knowledge
- Real-world programming challenges

The exercises range from basic syntax to advanced topics like OOP, data structures, file handling, databases, and more.

## How to Use This Repository

1. **Browse by Topic:** Navigate to folders based on what you want to practice
2. **Run Exercises:** Each Python file can be executed independently
3. **Learn from Solutions:** Review the code and understand the approach
4. **Modify and Experiment:** Try different solutions or extend the problems

### Running Exercises

```bash
# Navigate to a topic folder
cd Basics

# Run any exercise file
python exercise_name.py
```
